Created in IES JF as a school project

Written in: C++
Creator: TMS

Description: A simple notification program which allows you to know the IP from the using device without using get or requests, just using curl
Based: In my other IP notifiers projects and just wanted to write it in C++ as a fun fact

How to build: Using any compiler such us Cygwin or an online ide (Not recommended), you type in the cmd: g++ request.c++ -o request.exe
How to run: 

If you used cygwin(Windows) to compile: ./request.exe

If you used online ide just run

If you used linux to compile: ./request using the terminal
